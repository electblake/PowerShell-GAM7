# GAM7 PowerShell Module

GAM7 provides PowerShell cmdlets for Google Workspace mailbox lifecycle management, mailbox export, and encrypted GAM configuration backup/restore.

## Commands

- `Get-Mailbox`
- `Get-Mail`
- `Enable-Mailbox`
- `Disable-Mailbox`
- `Remove-Mailbox`
- `Export-Mailbox`
- `Invoke-BulkMailboxAction`
- `Debug-GAM`
- `New-GamEncryptionKey`
- `Backup-GamConfig`
- `Restore-GamConfig`
- `Export-GamAuthSecure`
- `Import-GamAuthSecure`
